### 开发说明

#### 启动项目

 ```sh
 // 下载项目
 git clone https://github.com/ZJU-CC98/CC98-PWA.git

 // 切换到开发分支
 git checkout dev

 // 安装依赖 (你也可以使用 yarn 或 cnpm)
 npm i

 // 启动开发服务器
 npm run dev
 ```

#### webpack 配置说明

开发配置允许开发者自定义

自定义的开发配置应放在在项目根目录下的 `.env/dev.js` 文件内

例子：

```js
// .env/dev.js

module.exports = {
  // 更改端口为 9000
  devServer: {
    port: 9000,
  },
}
```

不需要触发自动构建的提交请在 commit message 任意位置加上 `[skip ci]`
